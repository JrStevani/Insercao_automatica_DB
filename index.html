Projeto de Banco de Dados de Animes

Este projeto tem como objetivo principal criar e gerenciar um banco de dados MySQL para armazenar informações detalhadas sobre animes. Abaixo estão as principais funcionalidades implementadas neste projeto:

Banco de dados

  • Conexão com MySQL: Estabelecimento de uma conexão segura com o banco de dados MySQL utilizando a biblioteca mysql.connector.
  • Criação de Tabela: Verificação e criação da tabela animes no banco de dados, contendo colunas para armazenar o nome, descrição, linguagem, gêneros, imagem e temporadas/episódios dos animes.
  • Leitura de Arquivos JSON: Leitura e processamento de arquivos JSON contendo informações dos animes, como gêneros e temporadas/episódios.
  • Inserção de Dados: Inserção dos dados lidos nos arquivos JSON na tabela animes do banco de dados.
  • Contagem de Animes: Cálculo e exibição do total de animes inseridos na tabela.

Detalhamento das Funções

1. Conexão com o Banco de Dados

O projeto começa estabelecendo uma conexão com o banco de dados MySQL utilizando as credenciais fornecidas, como host, usuário, senha e nome do banco de dados.

2. Criação da Tabela de Animes

Em seguida, o código verifica se a tabela animes já existe no banco de dados. Caso contrário, ela é criada com colunas que incluem ID, nome, descrição, linguagem, gêneros, imagem, e temporadas/episódios.

3. Leitura e Processamento de Arquivos JSON

O script percorre diretórios específicos onde estão armazenados os arquivos JSON contendo informações sobre os animes. Esses arquivos são lidos, e os dados são extraídos e processados para serem inseridos no banco de dados.

4. Inserção de Dados no Banco de Dados

Após a leitura dos dados, eles são inseridos na tabela animes no banco de dados, utilizando comandos SQL de inserção.

5. Finalização e Contagem de Animes

Finalmente, a conexão com o banco de dados é fechada, e o total de animes inseridos é exibido.

Tecnologias Utilizadas

  • Python: Linguagem principal de programação utilizada para todo o desenvolvimento do projeto.
  • MySQL Connector: Biblioteca utilizada para a conexão e manipulação do banco de dados MySQL.
  • JSON: Módulo utilizado para a leitura e manipulação dos arquivos JSON.
  • Os: Módulo usado para operações relacionadas ao sistema operacional, como verificação da existência de arquivos e diretórios.
